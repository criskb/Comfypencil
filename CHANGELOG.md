# Changelog

## 2.0.0 - 2026-03-13

ComfyPencil 2.0 focuses on making the studio safer, more portable, and easier to learn without leaving the canvas.

- Added local recovery drafts backed by browser storage, with in-studio restore and dismiss actions when newer unsaved work is found
- Added portable brush preset library import/export from the Brush Library so presets can move between graphs and machines
- Added a built-in shortcut/help overlay plus a matching extension command for faster onboarding
- Improved save sequencing so newer changes are not silently treated as already saved when they land during an in-flight save
- Added a frontend setting to keep or disable local recovery drafts
- Updated package metadata and docs to reflect the 2.0 release surface
